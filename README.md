# From-Sound-to-Risk
From Sound to Risk: Streaming Audio Flags for Real-World Hazard Inference based on AI. JSAN MDPI
The system relies only on audio and is blind to visual data.

Read analyze audiosense0.docx for instructions.
Main points:
a) You may have to create two environments to avoid conflicts. Ours is myenv and audioapp
b) In verdict_rolling.py and verdict_rolling_llama3 one has to select the right prompt for the operational environment (shop, war, sports, monitoring of elders with dementia, urban environment). See prompts

Better to run it on a GPU. It needs many libs. It should be in the requirements but if a lib has slipped you may have to add it.
